# Smart Parking KSU

A smart parking detection system that helps students identify available parking spaces in campus parking lots.

The system uses computer vision and machine learning to classify parking spots as occupied or available from a camera feed.

## Project Goal

Parking at KSU can become chaotic during peak semesters; students often waste time searching for available spots. This project aims to build a system that detects parking spot availability and presents it through a simple web interface.

## Prototype Scope

Phase 1 prototype will:

- Use a video feed of a parking lot
- Detect individual parking spots
- Classify spots as occupied or empty
- Display parking availability on a website
